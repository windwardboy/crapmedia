# Bundled skins

CrapMedia ships with **two official skins** out of the box:

| Skin | ID | Background | Best for |
|------|-----|------------|----------|
| **Charcoal** | `charcoal` | Cool slate `#242628` | Default — daylight, OLED mounts |
| **Stone** | `stone` | Warm `#2A2826` | Long audiobooks, night driving |

Both use accent **`#00B87A`**. Users pick in **Settings → Appearance**.

## Preview

- **Charcoal:** `charcoal/preview.html` (side-by-side compare)
- Toggle `[data-theme="stone"]` on `<html>` to preview Stone alone

## Adding more

Community skins follow the same `theme.css` + `theme.json` contract. See [`docs/THEMING.md`](../../../docs/THEMING.md).
